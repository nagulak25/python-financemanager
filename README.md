# python-financemanager
A command-line-based Python application designed to help users record, store, analyze, and manage daily expenses. This project demonstrates core Python concepts such as object-oriented programming (OOP), file handling, error handling, modular code organization, and data persistence using CSV files. Ideal for beginners learning Python or for academic submissions.


# Features
Add and view expenses
Category-wise and monthly summaries
Data backup/restore
Input validation and error handling
# Installation
Clone the repo: git clone https://github.com/nagulak25/python-finance-manager.git
Navigate: cd python-finance-manager
Run: python main.py
Requires Python 3.x. No external dependencies.

# Usage
Run python main.py to start.
Follow the menu: Add expenses, view lists, generate reports.
Data saves to data/expenses.csv.

# Example:

Enter choice: 1
Enter amount: 1500
Enter category: Food
Enter date: 2024-01-15
Enter description: Grocery shopping
Expense added successfully!
